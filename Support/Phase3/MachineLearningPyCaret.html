Configuration du dataset¶

La fonction setup () initialise l'environnement dans pycaret et cre le pipeline de transformation pour prparer les donnes pour la modlisation et le dploiement. setup () doit tre appel avant d'excuter toute autre fonction dans pycaret. Il prend deux paramtres obligatoires: une trame de donnes pandas et le nom de la colonne cible. Tous les autres paramtres sont facultatifs et sont utiliss pour personnaliser le pipeline de prtraitement.

  1. Data Scale
  2. Target transformation

Tune model¶

Cette fonction rgle automatiquement les hyperparamtres d'un modle sur un espace de recherche prdfini et le note l'aide de la validation croise kfold. La sortie imprime une grille de score qui montre MAE, MSE, RMSE, R2, RMSLE et MAPE par pli.
